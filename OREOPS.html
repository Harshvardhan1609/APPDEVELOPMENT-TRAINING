Title: Schema and Data Definition

Description:

Data Model can be created both model to data or data to model vise versa. Only JSON format is allowed to create the model. Schema is auto generated and can be imported.

Note:

1. If any changes updated against the model properties like add,edit or delete means, the respective changes will not be updated automatically. So users should update the source and design against the changes. Else the execution might be a failure. Especially if a user having Firebase and SQLite connectors means this change is mandatory and recommended.

  • Properties

The session storage object to store data on a temporary basis, The data disappears when session ends i.e. when the application closed and removed from recent activity.For sqlite ,this option not applicable default cache

The local storage object to store data for your mobile on a permanent basis. That means the stored local data will be available until you uninstall or clear the storage in mobile devices.For sqlite ,this option not applicable default cache
